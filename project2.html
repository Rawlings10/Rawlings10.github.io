Echo Protocol

Action Horror Game

Project Description

Built in Unreal Engine.

This is a game in which player is trapped in a haunted house filled with zombies and dangerous creatures.

The main objective for the player is to find the exit from the house and rescue the hostages from the deadly creatures.

My Experience

This is a solo project, my very first game made with Unreal Engine and my second game as a game designer/programmer.

In this game, I encountered problems with the of mixamo animations and unreal animations and the use of AI for zombies, but after many trials and difficulties, this project now constitutes my knowledge base in the use of the Unreal engine..

I then learned the efficiency of using Unreal's blending space and animation blueprint, the most important aspect being the subsequent learning of the Unreal behavior tree which I then used in other projects to define AI instructions.

Constantly following this project makes me feel like this is where my incredible adventure began using unreal engine, and now, with my current knowledge, I see so many things I could have done better and didn't do back then.
