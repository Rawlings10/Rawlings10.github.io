Splink

Adventure Strategy Game

Project Description

Built in Unreal Engine.

This is a game where players must find all the elements needed to advance to the next level. There are five different types of elements, each with its own representation and meaning..

The orange element reprensent damage.

The green element reprensent HP.

The blue element is one of the three required element needed for progression (easy to find and less point).

The red element is also another element needed for progression (a little difficult to find and of a higher points than the blue).

The yellow element is the last element needed for progression (difficult to find and has the highest point).

My Experience

It was a solo project, my second game made with Unreal Engine.

This game allowed me to deepen my knowledge of level design, a key concept in the different stages of game development.

I was also able to improve my mastery of Unreal's tools, such as the sequencer, and above all, the menu design (user interface) enriched my experience in designing user interface.
