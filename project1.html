Shadow Ruins

Action Adventure Game

A third-person action adventure prototype focused on combat flow, traversal, and environmental storytelling.

Project Description

This project was built in Unreal Engine and focuses on responsive melee combat, enemy AI behavior, and level pacing.

Core systems implemented include animation state machines, hit detection, damage systems, and player ability mechanics.

Technology Used

Unreal Engine 5, C++, Blueprint, Blender
